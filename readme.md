# Modvodsk

Mods for the game "Podvodsk" by FourQuarters, using YYTK4 \& Aurie Framework.

### Components planned

* Explorer: In-Game variable explorer

  * Global variable explorer
  * Instance variable explorer
  * Run command
* Freeplay: Unlimited resources
* Enhanced Colors: Recolors the different types of buildings more distinctly
* Show depth: Shows the depth next to the score
* Better Settings: Sliders for music \& setting persistence
